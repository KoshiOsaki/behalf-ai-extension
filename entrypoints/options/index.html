Behalf AI 設定

APIキーの保存について

入力されたAPIキーは、お使いのブラウザの安全な保存領域（Chrome Storage Local API）に保存されます。

このデータは：

  • お使いのブラウザ内にのみ保存され、外部サーバーには送信されません
  • 他の拡張機能やWebサイトからアクセスすることはできません
  • この拡張機能を削除すると完全に削除されます
  • Behalf AI開発チームを含め、第三者がこのデータにアクセスすることはできません

APIキーは発言候補の生成時にのみ使用され、Google APIに直接送信されます。Behalf AIのサーバーにAPIキーが送信されたり保存されたりすることはありません。

✅ Notion 連携手順

  1. データベース作成
    例:「Behalf AI DB」を新規作成
  2. Integration 作成
    https://www.notion.com/my-integrations にアクセスし「+ New integration」
    任意の名前（例: Behalf AI integration）、Internal を選択
    読み取り・更新・挿入権限を付与し、Secret を取得
  3. Database ID 取得（詳細説明）
    作成したデータベースを クリックして開く
    右上「…」メニュー → 「ページとして開く（フルページ表示）」 をクリック
    （ビューが 1 ページとして表示される状態になります）
    ブラウザの URL を確認
    https://www.notion.so/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
    xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx の 32桁の英数字部分 が Database ID です。
    例: 1f6936f7c1308046a4ead0b155ed6f53
  4. Integration 共有設定
    右上「…」→ 接続 → 作成した Integration を選択し 追加
  5. Behalf AI 拡張機能設定
    Options ページで
    取得した Secret
    取得した Database ID
    を入力
